SPU Encode (SPU compression)

Purpose

Perform ADPCM compression on uncompressed audio data and convert to SPU stream file (*.ads) format.

Note: Only WAV files with straight PCM are supported. That is, the WAV file must have "fmt tag = 1".

Window

Input file (File List)

Displays a list of files to be processed. Files can be added to the list by dragging and dropping from Explorer or selecting files using the [Add] button.

Delete

Deletes the selected files from the file list.

Clear

Clears the file list.

Add

Displays the file selection dialog box (common dialog) to select files to be added to the input file list.

File Type

Selects the input file format.

Settings

If the Raw file format is selected, the sample format is set up.

AD-PCM (AD-PCM)

Performs ADPCM compression on the sample.

Output Folder

Specifies the folder in which files are output.

Browse

Displays a dialog (common dialog) for selecting an output folder.

Overwrite output file

If checked, pre-existing files will be overwritten by the output file.

Start

Start the operation.

->Operations screen

Parameter values

Interleave size 512 byte (PCM), 32 byte (ADPCM)
loop_start_block_address 0xffffffff (no loop)
loop_end_block_address 0xffffffff (no loop)

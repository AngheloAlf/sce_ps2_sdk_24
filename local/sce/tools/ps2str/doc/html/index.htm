ps2str stream converter tool

Purpose

Convert various media files to files used by the PlayStation 2.

Function

  1. Convert from MPEG video to IPU video
  2. Perform ADPCM compression and format conversion for uncompressed audio data
  3. Multiplex multiple elementary streams of video, audio, data, etc.
  4. Demultiplex multiplexed streams
  5. Display attributes of input data

command line format

General format

    ps2str [command] [options] [target file]

command section

Specify the feature to be used. Abbreviations can be used.

options section

The options for the different features will be described.

Available options vary according to feature.

target file section

Specify the target file.

Keywords are case sensitive.

The command section should be placed at the start of the command line parameters.

Example:

    ps2str demux -video test_stream.pss

Command

The following features can be used.

Command name Abbreviated form Function
convert c Convert from MPEG video to IPU video
encode e Perform ADPCM compression and format conversion of uncompressed audio data
mux m Multiplex multiple elementary streams such as video, audio, data
demux d Demultiplex multiplexed stream
info i Display attributes of input data
help h Help

GUI version

GUI operation is provided by running ps2str with no command line parameters or by directly running ps2strw.exe.

(Feature available only in Windows version)

Main window

A window is provided for each function, and windows are selected using the tab controls at the top.

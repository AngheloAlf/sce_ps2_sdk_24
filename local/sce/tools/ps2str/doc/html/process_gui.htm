Operations screen

If processing, display a screen indicating progress.

Window

Content of operation (Caption)

Displays what is being performed.

Processed file (File, Total Process)

Displays the name of the file being processed and the progress within the file list.

File processing status (Current Phase)

Displays the progress status of the file currently being processed.

Cancel

Cancels the operation.

Results screen

After the operation is finished, a log of results is displayed.

Save

Saves the results log text into a file.

Copy (Copy)

Copies the results log text to the clipboard.

Close

Closes the window.

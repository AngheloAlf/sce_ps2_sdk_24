Mux (multiplex streams)

Purpose

Multiplex data files and create pss file.

Window

Input file (File List)

Displays a list of files to be processed. Files can be added to the list by dragging and dropping from Explorer or selecting files using the [Add] button.

Multiplex instruction files are compatible with the command-line tools.

Add New

Create a new multiplex instruction file and add to the file list.

->Edit multiplex instruction file

After editing the instruction file, a dialog box (common dialog) for selecting a new filename is displayed.

Edit

Edit contents of selected multiplex instruction file.

->Edit multiplex instruction file

Add

Display file selection dialog box (common dialog) and select files to be added to the input file list.

Delete

Delete the selected files from the file list.

Clear

Clear the file list.

Output Folder

Specify the folder in which files are to be output.

Browse

Display the dialog box (common dialog) for selecting an output folder.

Overwrite output file

If checked, pre-existing files will be overwritten.

Start

Start the operation.

->Operations screen

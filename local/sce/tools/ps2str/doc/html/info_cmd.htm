info command

Purpose

Display attribute information of files.

Format

command section

info (Abbreviated form: i)

options section

Options Meaning
-v Provide detailed progress report
-pss Output pss file (MPEG2 program stream) data.
-ipu Output information about an ipu video stream file.
-ads Output information about an spu stream file.
-video Output information about an MPEG elementary stream file.
-wave Output information about a Windows WAVE audio file.
-aiff Output information about a Macintosh AIFF audio file.

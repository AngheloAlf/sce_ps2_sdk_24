IPU Convert (Convert IPU video)

Purpose

Convert MPEG1/2 video to IPU video.

Window

Input file (File List)

Displays a list of files to be processed. Files can be added to the list by dragging and dropping from Explorer or selecting files using the [Add] button.

Delete

Delete the selected files from the file list.

Clear (Clear)

Clear the file list.

Add

Display a file selection dialog box (common dialog) and select files to be added to the input file list.

Video elementary stream file (MPEG Video Elementary Stream)

Program stream file (MPEG2 Program Stream (PSS))

Select input file format.

Output Folder

Specify folder to which files are to be output.

Browse

Display a dialog (common dialog) for selecting an output folder.

Overwrite output file

If checked, any pre-existing file will be overwritten by the output file.

Start

Start the operation.

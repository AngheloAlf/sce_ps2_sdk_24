Demux (demultiplex stream)

Purpose

Demultiplex multiplexed stream.

Window

Input file (File List)

Display a list of files that can be processed. Files can be added to the list by draggin and dropping from Explorer or can be selected through the [Add] button.

Delete

Delete selected files from file list.

Clear

Clear the file list.

Add

Display the file selection dialog (common dialog) and select files to be added to the input file list.

Output stream (Stream)

Select the type, number of the stream to be extracted.

Output Folder

Specify folder in which files are to be output.

Browse

Display dialog for selecting output folder (common dialog).

Overwrite output file

If this is checked, existing files will be overwritten by the output file.

Start

Start the operation.

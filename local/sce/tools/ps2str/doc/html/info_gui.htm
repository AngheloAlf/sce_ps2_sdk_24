Info (Display information about file)

Purpose

Display attribute information of various files.

Window

Input file (File)

Specify file for which information is to be displayed.

Browse

Display a dialog box (common dialog) for selecting input file.

File Type

Select file type of file for which information is to be displayed.

Display (Show)

Display file information.

Tip

A file can be dragged and dropped into this window to have file information displayed.

File information

Display file information.

The information displayed will be the same as the command-line tool.

Save

Save file information text to file.

Tip

The window size can be changed.

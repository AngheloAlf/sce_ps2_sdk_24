Network configuration GUI library sample program
USB autoloader starter

<Description>
This program is an IOP module for enabling the USB autoloader used by sample programs. These programs show examples of how to use the network configuration GUI library (ntguicnf).

<Files>
	Makefile	:  Make file
	usbinit.c	:  Main program

<Usage>
	make clean
	make

	(*) Specifies usbinit.irx, which is created in this directory by /usr/local/sce/ee/sample/inet/ntguicnf/sample.c.

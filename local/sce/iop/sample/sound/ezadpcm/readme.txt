EzADPCM - Playing back BGM by streaming ADPCM data from disk

<Description>

This program shows how to perform playback by streaming an ADPCM waveform data file from the disk.

This sample is equivalent to the EE-side sample, and both function together. For details, refer to the EE-side readme file (ee/sample/sound/ezadpcm/readme.txt).

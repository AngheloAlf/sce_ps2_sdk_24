EzMIDI - Playing MIDI tunes and sound effects

<Description>
Tunes and sound effects are played using csl_hsyn, csl_midi, and csl_msin.

This sample program is associated with the sample program on the EE side, and the two programs work together. For more details, please see the readme file for the EE (ee/sample/sound/ezmidi/readme.txt).
